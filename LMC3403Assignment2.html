Step 1: Installing Git (~10 min)

Let's get Git on your machine!

  1. Download Git from git-scm.com.
  2. Run the installer and accept the default settings.

Step 2: Configure Git (~10 min)

Set up your identity for Git commits.

git config --global user.name "Your Name"
git config --global user.email "your_email@example.com"

Configuring Git with your name and email ensures every commit is tagged with your identity, making collaboration easier.

Step 3: Create Your First Repository (~15 min)

  • Sign up at GitHub.com.
  • Click "New Repository" and name it (e.g., "my-first-repo").

Step 4: Connect to GitHub (~15 min)

Link your local project to GitHub.

git init
git add .
git commit -m "First commit"
git remote add origin 
git push -u origin main

Tip: Replace with the URL from your GitHub repo.
