Step 1: Installing Git

  1. Download Git from git-scm.com
  2. Install Git (Accept defaults, recommended settings)

Step 2: Configure Git

git config --global user.name "Your Name"
git config --global user.email "your_email@example.com"

For more details, check out the Git documentation .

Step 3: Create your First GitHub Repository

  • Create an account at GitHub.com.
  • Click "New Repository" and follow the on-screen instructions.

Step 4: Connect Local Project to GitHub

git init
git add .
git commit -m "First commit"
git remote add origin <your_repo_URL>
git push -u origin main
