Step 1: Installing Git (~10 min)

Get Git running on your machine to start version control.

  1. Download from git-scm.com.
  2. Install with default settings.

Verify it worked:

git --version

Step 2: Configure Git (~10 min)

Set your identity for tracking changes.

git config --global user.name "Neil Goyal"
git config --global user.email "neil@example.com"

Why? This links your commits to you!

Check your setup:

git config --list

Step 3: Create a Repository (~15 min)

Store your code on GitHub!

  • Sign up at GitHub.com.
  • Click "New Repository" and name it "my-first-repo".

Tip: Make it public or private!

Step 4: Connect to GitHub (~15 min)

Push your local code online.

git init
git add .
git commit -m "First commit"
git remote add origin https://github.com/username/my-first-repo.git
git push -u origin main

Step 5: Branching Basics (~15 min)

Create branches to work on features safely.

git branch feature
git checkout feature
git add .
git commit -m "Add feature"
git checkout main
git merge feature

Why? Test changes without breaking main.
